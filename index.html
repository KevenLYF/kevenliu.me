Machine Learning: Learn Music

In this project, our team built a neural network on myRIO logic board by using labVIEW. The main feature of this neural network is to learn music (recoginize tones). By listening to the tones or chords, the program will tell which tone or chord is being played.

The recognition requires two modes: a "training" mode and a "inference" mode. In "training" mode, we feed the dataset into the feedback network, and it will update it's parameters in the backpropagation algorithm which we wrote in labVIEW. The training process will finish once it's output posibility is at the closet point to "1". In "inference" mode, we will play tones and chords, and the program will tell us what it just heard. The accuracy of our program exceed 95%.

Cross Platform TA Management System

Note: as mentioned by my teammate, the new McGill MyFuture website may "copied" my website design :) link to myFuture



We developed a TA management System in 3 platfroms: a Java program in PC, an Android app and a website which is written in HTML, CSS and PHP. The system enables students to apply for a TA position, or a professor to review all the TA applications.


The system was developed under standard software development process. First we used Umple modeling tool to model the system and then we used git to collaborate everyone's work. The development process was also tracked by using Gantt Chart. After the developemnt was done, we performed Unit Test and Integration Test by using respective frameworks (JUnit, PHP Unit, XDebug). The CI and building process was monitored by using Travis CI.

AM Reciever

The AM reciever, which commonly known as radio is a device that takes baseband frequency as input and output audio information. We made a radio station from the most original materials: wires, resistors, capacitors, BJTs and MOSFETs. The development process is caculation - simulation - assembling. We used knowledge learned from microelectronic classes to calculate parameters, and simulated the whole design in PSPICE.
